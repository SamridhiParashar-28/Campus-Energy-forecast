// about the project

What is WattWise?

WattWise is a web-based energy monitoring platform developed as part of a final-year engineering project. It targets university campuses — specifically hostels and academic blocks — where energy usage is high but poorly tracked.

The system ingests daily meter readings via CSV upload, runs a hybrid LSTM + XGBoost forecasting model, and surfaces insights through an interactive real-time dashboard. Anomalies are flagged automatically when usage deviates from predicted baselines.

Built on a Node.js + Express backend with a vanilla JavaScript frontend, the project is designed to be lightweight, deployable without cloud infrastructure, and easy to extend with a full Python ML pipeline.

CSV Data Upload Upload meter readings per block, room and appliance. Supports the full campus dataset format.
Live Usage Monitoring Zone-level real-time consumption view across all 5 campus blocks, refreshing automatically.
Short-Term Forecasting LSTM + XGBoost hybrid model predicts next-day kWh per block with 94.2% accuracy.
Anomaly Detection Automatically flags unusual spikes and baseline deviations with severity ratings.
Billing & Cost Simulation Per-block and per-appliance cost estimates at configurable tariff rates.
